Indice de masa corporal

El índice de masa corporal (IMC) es una medida que se utiliza para evaluar si una persona tiene un peso saludable en relación con su altura. Para calcularlo, sigue estos pasos: Divide tu peso (en kilogramos) entre tu altura (en metros). Divide el resultado nuevamente entre tu altura para obtener tu IMC. La fórmula del IMC se expresa como: IMC=altura2peso​

Porcentaje de grasa corporal

Fórmula de Deurenberg: La fórmula de Deurenberg es más sencilla y se basa en el IMC, la edad, el sexo y otros factores. Para calcular el porcentaje de grasa corporal utilizando esta fórmula, sigue estos pasos:

Para hombres:

Porcentaje de grasa corporal=(1.20×IMC)+(0.23×Edad)−(10.8×Genero)−5.4

Para mujeres

Para mujeres: Porcentaje de grasa corporal=(1.20×IMC)+(0.23×Edad)−5.4

Donde: IMC es el Índice de Masa Corporal calculado previamente. Edad es la edad en años. Género: Hombre = 1, Mujer = 2
